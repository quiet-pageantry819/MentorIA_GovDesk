# Getting Started with the Prototype

*(Note: The active source code for the MVP is currently under development. This document outlines the expected interaction patterns for when the prototype is released for pilot testing.)*

## Setting Up for Local Development
Instructions will be provided here for spinning up the local RAG engine, the agentic orchestration layer, and the conversational frontend interface.

## Running the Mock Environment
The prototype will include a suite of mock municipal APIs (e.g., a mock ticketing system and a mock appointment scheduler) to allow developers and civic tech partners to test agentic reasoning without needing a real government integration.

## Analytic Dashboard
The first release will include a read-only Grafana/Kibana dashboard illustrating simulated public service demand metrics generated by the agent interactions.
